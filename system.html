앱링커 - AppLinker 스마트폰으로 PC/포스/키오스크 등의
디바이스를 제어 할수 있으며
멀티탭과 연동하여 전원 제어 기능을 지원합니다.

주요 기능

- 내 컴퓨터 화면을 원격 제어

- 설정된 스케줄 시간에 컴퓨터를 재부팅

- 멀티탭과 연동하여 전원 제어 기능을 지원

퀵가이드

1. 앱링커 설치후 회원 가입을 합니다.

2. 제어 하려는 기기에 앱링커 클라이언트를 설치 합니다.

3. 클라이언트 설치하면 앱에 단말기가 등록 처리 됩니다.

4. 앱에 등록된 단말기를 통해 원격제어를 할수 있습니다.

상세 설명은 www.zeonix.co.kr에서 확인 바랍니다.
